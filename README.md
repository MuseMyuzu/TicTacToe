# 三目並べ（マルバツゲーム）TicTacToe
 ChatGPTを利用して Tic Tac Toe を作ってみました。
 三目並べ、マルバツゲームとも呼ばれています。
 
 〇や×を交互に置いていき、先にタテヨコナナメのどれか一列並べた方の勝ちというゲームです。
